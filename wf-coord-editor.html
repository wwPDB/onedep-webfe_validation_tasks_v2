  Instructions
  • Coordinates are organized according to PDB chain IDs. For each PDB chain ID, it contains polymer chain, nonpolymer(s) and water (if they exist).
  • For each atom, atom name, atom type, altLoc and occupancy fields can be edited by clicking the values in these two fields. Atom can be deleted by changing deletion flag from "N" to "Y".
  • For each residue, number, insertion code and altLoc (applying to all atoms in the residue) fields can be edited.
  • For nonpolymer and water residue, the PDB chain ID field can also be edited.
  • "Renumbering Polymer/Nonpolymer/Water Starting from:" boxes are for global renumbering.
  • For polymer chain, "Renumbering Polymer using Range:" box is for unusual numbering schema (Add instruction and example).
  • "Rename PDB chain ID:" box is for renaming PDB chain ID (applying to all polymer, nonpolymer(s) and water (if they exist)).
  • Update is done in the following order: atom level change, residue level change, global renumbering and renaming PDB chain ID.
