Validation Tasks Help Information

Upload

Provides upload option of model coordinate file. Various flavors of PDBx/mmCIF format are supported. Files with non-standard file names will be renamed by project conventions using the placeholder D_0000000000 data set identifier.

Dataset may also be identified by data set id code. The latest files in the archive corresponding to the identifier will be be downloaded.

Validation Report

Create the wwPDB validation report and associated data and image files.

Check Reports

Create check reports for dictionary compliance, geometry, special positions, miscellaneous consistency checks, and DCC SF checks (incl. refinement).

Viewer 3D

View input model coordinate data using the JMol applet or JSMol.

Download

Download copies of the updated model file containing calculated annotations, coordinates of any calculated assemblies, analysis file created by the Pisa program, site environment file from getsite, and the log files from each of the annotation task calculations.
